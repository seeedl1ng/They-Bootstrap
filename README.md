# They Bootstrap

Available: only 1.12.2 Cleanroom пока что 

Мод проверяет наличие модов,  ресурспаков и шейдеров, и если ни одного нету, то загружает их по ссылкам из папки .index (в mods, resourcepack и shaders)

Это лучше чем .mcpack и curseforge
потому что поддерживает скачивание с любых платформ (включая github?)
